AttackAlert Documentation
=========================

AttackAlert is a comprehensive Discord bot for Clash of Clans war monitoring and reminders. It supports multiple Discord servers with complete guild-specific data isolation, hybrid database/JSON storage, and advanced war tracking features including CWL (Clan War League) support.

**Key Features:**

- **Multi-Server Support**: Complete data isolation between Discord servers
- **Advanced War Tracking**: Normal wars and CWL with real-time monitoring
- **Smart Reminder System**: Timed reminders (1hr, 30min, 15min) with targeted notifications
- **Account Management**: Link multiple CoC accounts to Discord users per server
- **Security First**: Guild isolation and clan conflict prevention
- **Hybrid Storage**: Database-first with JSON fallback for maximum reliability
- **4 Clan Limit**: Each Discord server can monitor up to 4 clans simultaneously
